# 抖音Cookies示例文件
#
# 🚀 使用方法：
# 1. 登录抖音网页版 (www.douyin.com)
# 2. 按F12 → Network → 刷新页面 → 点击任意请求 → Headers → Cookie
# 3. 复制完整的Cookie字符串粘贴到下面
# 4. 将此文件重命名为 douyin.txt
#
# 📋 格式示例：
# sessionid=your_session_value; sid_guard=your_sid_guard; uid_tt=your_uid; other_cookies=values
#
# ⚠️ 注意：删除所有注释行，只保留cookie字符串

# 请在此行下方粘贴您的完整cookies字符串
